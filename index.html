Proyectos

Encriptador de texto

Este proyecto es una aplicación web diseñada para encriptar y desencriptar mensajes de texto, proporcionando una interfaz sencilla y elegante. La aplicación está desarrollada utilizando HTML, CSS, y JavaScript, con el apoyo de la librería FontAwesome para iconos.

Lista de tareas (To-do App)

Esta lista de tareas es una app web con el fin de agregar tareas, marcar tareas como terminadas o pendientes, eliminar todas las tareas o una en especifica, actualizar el nombre de una tarea. Tiene la posibilidad de guardar las tareas en el cache del navegador por lo que si recargas la pagina conservara el ultimo estado de las tareas. Esta app es ideal para organizar tus actividades del dia. Esta aplicacion esta hecha con HTML, CSS, Javascript, Chakra UI y React.js. Cuenta tambien con un boton para modo nocturno o dia.

Chat con Rachet Web Sockets

Este proyecto es un Chat en el cual puedes comunicarte a traves de mensajes de textos con otros usuarios en tiempo real. Pueden conectarse a la sala muchos usuarios. Pueden entrar con un nombre personalizado o generico. Esta aplicacion esta desarrollada en PHP con la libreria de Sockets Ratchet Web Sockets que es la que permite la comunicación bidireccional entre cliente y servidor. Se uso tecnologías como HTML, CSS, JavaScript, PHP, APACHE, Composer.

Alura Geek

Este proyecto es una tienda de productos en la cual puedes agregar, eliminar y buscar productos. Realizada con las tecnologias de HTMl, CSS, JS, y json-server.
